# Asset Manifest

This workspace copies the generated LaTeX assets from `arxiv_report/paper/latex_assets/`.

## Figures

- `figures/generated/overall_pipeline.pdf`: main workflow framework figure used in the body.
- `figures/generated/inner_loop.pdf`: inner-loop construction-time correction figure used in the body.
- `figures/generated/outer_loop.pdf`: outer-loop validation-time improvement figure used in the body.
- `figures/generated/overall_signed_relative_change.pdf`: direction-normalized overall metric changes.
- `figures/generated/spatialedit_external_relative_change.pdf`: external base/fal/ours comparison, direction-normalized against the base model.
- `figures/generated/spatialedit_vlm_score_bars.pdf`: external VLM view and consistency score comparison.
- `figures/generated/overall_metric_values_grouped.pdf`: absolute exp5 vs R1 metric values.
- `figures/generated/per_angle_diagnostic_deltas.pdf`: per-angle deltas for Score_view, PSNR, and DINO.
- `figures/generated/vie_per_angle_view_consistency.pdf`: per-angle VIEScore view and consistency bars.
- `figures/generated/dataset_scaling_and_quality.pdf`: dataset size and R1 hybrid-score diagnosis.
- `figures/generated/r1_bootstrap_hybrid_convergence.pdf`: R1 hybrid-score trajectories and gate threshold.
- `figures/generated/r1_augmentation_composition.pdf`: R1 category and target-angle composition.
- `figures/generated/r1_issue_tag_counts.pdf`: VLM/CV issue tag counts.
- `figures/generated/qualitative_rotation_grid.pdf`: GT-only rotation grid; not used as a main comparison figure.
